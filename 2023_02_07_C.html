Algorithm Study Note

C for 루프 — 4의 배수 카운팅 (1 ~ 2023)

C · for loop · % 나머지 · 배수 카운팅 · Tracing
Q문제 코드
C 
#include <stdio.h>

int main(){

    int c = 0;

    for(int i = 1; i <= 2023; i++) {
        if(i % 4 == 0) c++;   // 4의 배수이면 c 증가
    }

    printf("%d", c);

}

위 코드의 출력값은 무엇인가?

printf("%d", c) 출력 결과
505
1 ~ 2023 중 4의 배수: 4, 8, 12, … , 2020
⌊2023 ÷ 4⌋ = 505
① 문제 유형

for 루프로 1부터 2023까지 순회하면서 나머지 연산(%)으로 4의 배수 개수를 세는 카운팅 추적 문제입니다.

② 핵심 수학 공식
1 ~ N 범위의 k의 배수 개수 = ⌊ N ÷ k ⌋
↓
1 ~ 2023 에서 4의 배수 개수
= ⌊ 2023 ÷ 4 ⌋
= ⌊ 505.75 ⌋
= 505
③ 범위 경계 확인
확인 항목 계산 결과
첫 번째 4의 배수 4 × 1 = 4 ✓ 범위 내 (4 ≥ 1)
마지막 4의 배수 4 × 505 = 2020 ✓ 범위 내 (2020 ≤ 2023)
그 다음 배수 4 × 506 = 2024 ✗ 범위 초과 (2024 > 2023)
총 개수 ⌊2023 ÷ 4⌋ 505
④ 루프 샘플 추적 (앞 10개 + 끝 2개)
i i % 4 조건 c

⑤ 핵심 포인트
  • 🔢 1 ~ N 범위의 k 배수 개수 = ⌊N / k⌋ (정수 나눗셈)
  • 〰️ i % 4 == 0은 i가 4로 나누어 떨어질 때만 참 → 4의 배수
  • 🏁 2023 범위 내 마지막 4의 배수는 2020 (4 × 505), 2024는 범위 초과
