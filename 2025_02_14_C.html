C 이중 포인터 & 구조체 배열 추적

C · Struct · Double Pointer · Array · Dereference · Tracing
Q문제 코드
C 
#include <stdio.h>

struct dat {
    int x;
    int y;
};

int main() {
    struct dat a[] = {{1,2}, {3,4}, {5,6}};
    struct dat*  ptr  = a;         // ptr → a[0]
    struct dat** pptr = &ptr;      // pptr → ptr

    (*pptr)[1] = (*pptr)[2];     // a[1] = a[2]
    printf("%d 그리고 %d", a[1].x, a[1].y);

    return 0;
}

위 코드의 출력값은 무엇인가?

printf 출력 결과
5 그리고 6
(*pptr)[1] = (*pptr)[2] → a[1] = a[2] = {5,6}
① 문제 유형

이중 포인터(double pointer)를 역참조하여 구조체 배열 원소를 복사하는 결과를 추적하는 문제입니다.


  • 🔷*pptr — pptr을 역참조하면 ptr이 나옵니다. ptr은 배열 a의 시작 주소입니다.
  • 🔷(*pptr)[n] — ptr[n]과 동일 = a[n] (구조체 배열 인덱싱)
  • 🔷구조체 배열 대입 a[1] = a[2] — 구조체 전체(x, y 모두)가 복사됩니다.
② 포인터 연결 구조
③ (*pptr)[1] 연쇄 역참조 분해
1 pptr → ptr의 주소 이중 포인터: ptr을 가리킴
2 *pptr → ptr 역참조 → ptr 자체 (= a의 시작 주소)
3 (*pptr)[1] → ptr[1] = a[1] 인덱스 1 = 두 번째 구조체
4 (*pptr)[2] → ptr[2] = a[2] = {5,6} 인덱스 2 = 세 번째 구조체
5 (*pptr)[1] = (*pptr)[2] → a[1] = {5, 6} 구조체 전체 복사 (x=5, y=6)
④ 배열 a[] 상태 변화
시점 a[0].x / a[0].y a[1].x / a[1].y a[2].x / a[2].y
초기 1 / 2 3 / 4 5 / 6
대입 후 1 / 2 (불변) 5 / 6 ← 변경! 5 / 6 (원본)
출력 — a[1].x=5, a[1].y=6 —
⑤ 핵심 주의 포인트
  • 🔗 pptr = &ptr → *pptr == ptr, **pptr == *ptr == a[0]. 역참조를 단계별로 분리하세요.
  • 📐 (*pptr)[n]은 ptr[n] = a[n]과 완전히 동일합니다. 괄호가 역참조 우선순위를 확보합니다.
  • 📋 C에서 구조체 대입 a[1] = a[2]는 멤버 전체를 값 복사합니다. x, y 모두 5, 6으로 덮어씁니다.
  • 🖨️ printf("%d 그리고 %d", a[1].x, a[1].y) → 5 그리고 6
