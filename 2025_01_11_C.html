C 동적 2D 배열 인덱스 매핑 & 교대 부호 합산 추적

C · malloc · 2D Array · Index Formula · Modulo · Alternating Sum · Tracing
Q문제 코드
C 
#include <stdio.h>
#include <stdlib.h>

void set(int** arr, int* data, int rows, int cols) {
    for (int i = 0; i < rows * cols; ++i) {
        arr[((i + 1) / rows) % rows][(i + 1) % cols] = data[i];
    }
}

int main() {
    int rows = 3, cols = 3, sum = 0;
    int data[] = {5,2,7,4,1,8,3,6,9};
    int** arr = (int**) malloc(sizeof(int*) * rows);
    for (int i = 0; i < cols; i++)
        arr[i] = (int*) malloc(sizeof(int) * cols);

    set(arr, data, rows, cols);

    for (int i = 0; i < rows * cols; i++) {
        sum += arr[i / rows][i % cols] * (i % 2 == 0 ? 1 : -1);
    }
    for (int i = 0; i < rows; i++) free(arr[i]);
    free(arr);
    printf("%d", sum);
}

① 문제 유형

복잡한 인덱스 수식으로 data[]를 2D 배열에 배치한 뒤, 또 다른 인덱스 수식으로 교대 부호 합을 구하는 문제입니다.


  • 🔷set() 배치 수식: row = ((i+1)/rows)%rows, col = (i+1)%cols — i=0~8의 각 값을 계산해야 합니다.
  • 🔷합산 인덱스: arr[i/rows][i%cols] — 읽는 순서가 배치 순서와 다릅니다. i=0~8을 각각 계산합니다.
  • 🔷교대 부호: i%2==0 ? +1 : -1 — 짝수 인덱스는 덧셈, 홀수 인덱스는 뺄셈입니다.
② set() — data[i] → arr[row][col] 배치 추적
i data[i] i+1 (i+1)/3 row = /3 %3 col = (i+1)%3 배치
0 5 1 0 0%3 = 0 1%3 = 1 arr[0][1] = 5

2 7 3 1 1%3 = 1 3%3 = 0 arr[1][0] = 7
3 4 4 1 1%3 = 1 4%3 = 1 arr[1][1] = 4

⚠️ i=8 (마지막)에서 row=0, col=0 → arr[0][0]=9 — 맨 처음 칸이 마지막에 채워집니다!

⑥ 핵심 주의 포인트
  • ⚠️ i=8 마지막에 arr[0][0]=9 배치: ((8+1)/3)%3 = (3)%3 = 0, (8+1)%3 = 0 → arr[0][0]. 맨 처음 칸이 마지막에 채워지는 것이 핵심 함정입니다.
  • 📐 배치 수식 vs 읽기 수식이 다릅니다. set()의 ((i+1)/rows)%rows와 합산의 i/rows는 전혀 다른 순서로 접근합니다.
  • ➕ 합산 인덱스 i/rows는 정수 나눗셈으로 행을 결정합니다. i=0~2→행0, i=3~5→행1, i=6~8→행2.
