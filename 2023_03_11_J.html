Algorithm Study Note

Java 다형성 + 재귀 오버라이딩 추적 (num-2 vs num-3)

Java · Polymorphism · Dynamic Dispatch · Recursion · Overriding · Tracing
Q문제 코드
Java 
class Parent {
    int compute(int num) {
        if (num <= 1) return num;
        return compute(num - 1) + compute(num - 2);  // 피보나치 형태
    }
}
class Child extends Parent {
    int compute(int num) {
        if (num <= 1) return num;
        return compute(num - 1) + compute(num - 3);  // ★ -2 아닌 -3!
    }
}
public class main {
    public static void main(String args[]) {
        Parent obj = new Child();  // 실제 객체: Child
        System.out.print(obj.compute(7));
    }
}

위 코드의 출력값은 무엇인가?

System.out.print() 출력 결과
2
동적 디스패치 → Child.compute(7) 호출
compute(num-3) 때문에 음수 진입 → 대부분 0으로 수렴
① 문제 유형 & 핵심 함정

동적 디스패치로 Child.compute()가 호출되며, Parent의 피보나치(num-2)와 달리 Child는 num-3을 사용하여 음수 영역까지 진입하는 재귀 추적 문제입니다.

★ 두 가지 함정
1. Parent obj = new Child() → Child.compute() 실행 (동적 디스패치)
2. num - 3 때문에 num이 음수가 될 수 있음 → 기저조건 num <= 1에서 음수도 return num
② Parent.compute vs Child.compute 비교
Parent (호출 안 됨)
return compute(num-1)
     + compute(num-2)
→ 피보나치 수열
Child (실제 실행)
return compute(num-1)
     + compute(num-3)
→ 음수 진입 가능!
③ 기저 조건 분석 — 음수도 처리됨!
if (num <= 1) return num;
↑ num=1 → return 1 ↑ num=0 → return 0 ↑ num=-1 → return -1 (음수도 그대로 반환!)
④ Child.compute() 단계별 계산 (바텀업)
호출 조건 계산 반환값
compute(-1) -1 ≤ 1 → 기저 — -1
compute(0) 0 ≤ 1 → 기저 — 0
compute(1) 1 ≤ 1 → 기저 — 1
compute(2) 2 > 1 compute(1) + compute(-1) = 1 + (-1) 0
compute(3) 3 > 1 compute(2) + compute(0) = 0 + 0 0
compute(4) 4 > 1 compute(3) + compute(1) = 0 + 1 1
compute(5) 5 > 1 compute(4) + compute(2) = 1 + 0 1
compute(6) 6 > 1 compute(5) + compute(3) = 1 + 0 1
compute(7) 7 > 1 compute(6) + compute(4) = 1 + 1 2
⑤ compute(2) 핵심 — 음수 진입
compute(2)
= compute(2-1) + compute(2-3)
= compute(1) + compute(-1)
= 1 + (-1) ← -1 <= 1 이므로 return -1
= 0
⑥ 핵심 주의 포인트
  • 🎯 동적 디스패치: Parent obj = new Child()이므로 Child.compute()가 실행됩니다. Parent.compute()는 단 한 번도 호출되지 않습니다.
  • ⚠️ 음수 기저 조건: num <= 1은 음수도 포함합니다. compute(-1)은 -1을 반환합니다.
  • 🔢 compute(2) = 1+(-1) = 0이 핵심 — 이것이 0으로 수렴하는 출발점입니다.
  • 🖨️ 최종 출력: 2
System.out.print() 출력 결과
2
Child.compute(7) = compute(6)+compute(4) = 1+1 = 2
compute(2)=1+(-1)=0 이 핵심 — num-3으로 음수 진입
