Python 리스트 제자리 역순 + 슬라이싱 합산 추적

Python · List · In-place Reverse · Negative Index · Slice · sum · Tracing
Q문제 코드
Python 
def test(lst):
    for i in range(len(lst) // 2):           # i = 0, 1, 2
        lst[i], lst[-i-1] = lst[-i-1], lst[i]  # 앞↔뒤 동시 스왑

ls = [1, 2, 3, 4, 5, 6]
test(ls)
print(sum(ls[::2]) - sum(ls[1::2]))

위 코드의 출력값은 무엇인가?

print() 출력 결과
3
역순 후 [6,5,4,3,2,1] → sum([6,4,2]) - sum([5,3,1]) = 12 - 9 = 3
① 문제 유형

음수 인덱스를 이용한 제자리(in-place) 리스트 역순 후, 파이썬 슬라이싱으로 짝수/홀수 인덱스 합의 차를 구하는 문제입니다.


  • 🔷제자리 역순: lst[i], lst[-i-1] = lst[-i-1], lst[i] — i=0~2 동안 앞·뒤 원소를 쌍으로 스왑하여 전체를 뒤집습니다.
  • 🔷슬라이싱 합산: ls[::2]는 인덱스 0,2,4 (짝수 인덱스), ls[1::2]는 인덱스 1,3,5 (홀수 인덱스)의 원소를 선택합니다.
② test() 스왑 루프 — 단계별 추적
i lst[i] ↔ lst[-i-1] 스왑 대상 스왑 후 리스트
0 lst[0] ↔ lst[-1] 1 ↔ 6 [6,2,3,4,5,1]
1 lst[1] ↔ lst[-2] 2 ↔ 5 [6,5,3,4,2,1]
2 lst[2] ↔ lst[-3] 3 ↔ 4 [6,5,4,3,2,1]

range(6 // 2) = range(3) → i=0,1,2 → 완전한 역순 완성

④ 슬라이싱 합산 계산
슬라이스 선택 인덱스 선택 원소 sum()
ls[::2] 0, 2, 4 [6, 4, 2] 6+4+2 = 12
ls[1::2] 1, 3, 5 [5, 3, 1] 5+3+1 = 9
sum(ls[::2]) − sum(ls[1::2]) = 12 − 9 = 3
⑤ 핵심 주의 포인트
  • 🔄 in-place 역순 원리: lst[i], lst[-i-1] = lst[-i-1], lst[i] — Python의 다중 대입(동시 평가)이므로 임시 변수 없이 안전하게 스왑됩니다.
  • 📌 range(len(lst) // 2): 길이 6 → range(3) → i=0,1,2. 길이가 짝수이므로 정확히 절반씩 스왑하여 완전 역순이 됩니다.
  • ✂️ ls[::2]는 step=2, 0부터 → 인덱스 0,2,4 선택. ls[1::2]는 step=2, 1부터 → 인덱스 1,3,5 선택.
