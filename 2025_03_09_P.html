Algorithm Study Note

Python enumerate & 딕셔너리 구성 추적

Python · enumerate · dict · sum · len · Tracing
Q문제 코드
Python 
data = [
    [3, 5, 2, 4, 1],
    [4, 5, 1],
    [4, 4, 1, 5, 4],
    [4, 5]
]

result = {}

for index, lis in enumerate(data):
    list_sum = sum(lis)       # 리스트 합계
    list_len = len(lis)       # 리스트 길이

    result[index] = (list_sum, list_len)   # 튜플로 저장

print(result)

위 코드의 출력값은 무엇인가?

print(result) 출력 결과
{0: (15, 5), 1: (10, 3), 2: (18, 5), 3: (9, 2)}
각 리스트의 (합계, 길이) 튜플을 인덱스 키로 저장
① 문제 유형

enumerate()로 인덱스와 원소를 동시에 순회하며 딕셔너리를 구성하는 출력값 추적 문제입니다.


  • 🔷enumerate(iterable) — 반복할 때 (인덱스, 값) 쌍을 함께 반환합니다.
  • 🔷sum(list) / len(list) — 리스트의 합계와 길이를 구하는 내장 함수입니다.
  • 🔷튜플로 저장 — result[index] = (sum, len) 형태로 딕셔너리에 저장합니다.
② for 루프 단계별 추적
index lis list_sum (sum) list_len (len) result[index]
0 [3, 5, 2, 4, 1] 3+5+2+4+1 = 15 5 (15, 5)
1 [4, 5, 1] 4+5+1 = 10 3 (10, 3)
2 [4, 4, 1, 5, 4] 4+4+1+5+4 = 18 5 (18, 5)
3 [4, 5] 4+5 = 9 2 (9, 2)
최종 result = {0:(15,5), 1:(10,3), 2:(18,5), 3:(9,2)}

④ 핵심 정리
  • 📌 enumerate(data)는 (0, data[0]), (1, data[1]), … 순서로 인덱스와 값을 함께 반환합니다.
  • 📦 result[index] = (list_sum, list_len) — 딕셔너리의 값으로 튜플이 저장됩니다. 괄호가 튜플을 만듭니다.
  • 🎨 출력 형식은 Python 딕셔너리 표준 표현식: {key: value, ...} 형태로 출력됩니다.
  • 🔵 딕셔너리 키는 0, 1, 2, 3의 정수이고, 값은 (합계, 길이)의 튜플입니다.
