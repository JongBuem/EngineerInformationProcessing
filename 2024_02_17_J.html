Algorithm Study Note

Java 재귀 + seen 배열 중복 제거 역순 추적

Java · Recursion · boolean[] seen · Deduplication · String · Tracing
Q문제 코드
Java 
public class Main {
    public static String rf(String str, int index, boolean[] seen) {
        if (index < 0) return "";           // 기저 조건
        char c = str.charAt(index);
        String result = rf(str, index-1, seen); // ① 먼저 재귀 (왼쪽 먼저 처리)
        if (!seen[c]) {                           // ② 돌아올 때 중복 확인
            seen[c] = true;
            return c + result;                   // 앞에 붙임
        }
        return result;
    }

    public static void main(String[] args) {
        String str = "abacabcd";
        int length = str.length();            // 8
        boolean[] seen = new boolean[256];
        System.out.print(rf(str, length-1, seen));
    }
}

① 문제 유형

재귀로 왼쪽부터 먼저 처리(seen 마킹)하고, 복귀하면서 중복을 제외한 문자를 앞에 이어붙여 역순 고유 문자열을 만드는 문제입니다.


  • ⚠️핵심 순서: 재귀 먼저, 확인 나중. result = rf(..., index-1, ...)를 먼저 호출하므로 왼쪽 인덱스의 문자가 먼저 seen에 마킹됩니다.
  • 🔷복귀 시 !seen[c] — 현재 c가 아직 seen에 없으면 c + result(앞에 붙임)로 반환합니다.
  • 🔷결과: 첫 등장 순서(a, b, c, d)가 역순으로 출력됩니다.

⑥ 핵심 주의 포인트
  • 🔄 재귀 먼저, 확인 나중: result = rf(str, index-1, seen)을 먼저 호출하므로 왼쪽(작은 인덱스)의 문자가 먼저 seen에 마킹됩니다. 중복 판단은 항상 왼쪽 기준입니다.
  • 🔤 c + result: 새 문자를 앞에 붙입니다. 복귀 순서는 인덱스 오름차순(0→7)이므로, 나중에 복귀할수록 더 앞에 붙어 역순이 됩니다.
  • 💡 결국 첫 등장 순서(a→b→c→d)를 역순으로 출력하는 알고리즘입니다: dcba
