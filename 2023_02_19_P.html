Python 문자열 슬라이싱 & 연결 추적

Python · String Slicing · [start:end] · [start:] · 인덱스 추적 · Tracing
Q문제 코드
Python 
a = "engineer information processing"
b = a[:3]        # 인덱스 0 ~ 2
c = a[4:6]       # 인덱스 4 ~ 5
d = a[28:]       # 인덱스 28 ~ 끝
e = b + c + d
print(e)

위 코드의 출력값은 무엇인가?

print(e) 출력 결과
engneing
b="eng" + c="ne" + d="ing"
a[28:] → 인덱스 28, 29, 30 = "ing"
① 문제 유형

Python 문자열 슬라이싱 [start:end]와 끝까지 자르는 [start:] 규칙을 적용해 문자열을 연결하는 추적 문제입니다.

⑤ 핵심 포인트
  • ✂️ a[start:end]에서 end는 포함되지 않습니다. a[4:6]은 인덱스 4, 5만.
  • 🔚 a[28:]처럼 end를 생략하면 끝까지 슬라이싱합니다. "processing"의 마지막 3글자 = "ing".
  • 📏 문자열 전체 길이: "engineer information processing" = 31자 (인덱스 0~30)
