Python 집합(set) 연산 — add / remove / update 추적

Python · set · add() · remove() · update() · 중복 불가 · 순서 없음
Q문제 코드
Python 
a = {'한국', '중국', '일본'}
a.add('베트남')           # 새 원소 추가
a.add('중국')             # 이미 존재 → 무시
a.remove('일본')          # 원소 제거
a.update({'홍콩', '한국', '태국'})  # 집합 합치기 (중복 무시)
print(a)

위 코드의 최종 집합 a에 포함된 원소는?

print(a) — 포함된 원소 (5개)
{'한국', '중국', '베트남', '홍콩', '태국'}
⚠ set은 순서가 없으므로 출력 순서는 실행마다 다를 수 있음
추가: 베트남·홍콩·태국 / 중복 무시: 중국·한국 / 제거: 일본
① 문제 유형

Python set(집합)의 add / remove / update 메서드를 순서대로 적용하고 최종 원소 집합을 구하는 추적 문제입니다. set의 핵심 특성인 중복 불가와 순서 없음을 이해해야 합니다.

update(S) 집합 S의 원소 전부 추가 중복은 무시 새 원소만 추가

④ 단계별 추적표
단계 코드 동작 원소 수
초기 {'한국','중국','일본'} 집합 생성 3
1 add('베트남') 추가 베트남 없음 → 삽입 4
2 add('중국') 무시 이미 존재 → 변화 없음 4
3 remove('일본') 제거 일본 삭제 3
4 update({'홍콩','한국','태국'}) 합집합 홍콩·태국 추가, 한국 무시 5
최종 print(a) {'한국', '중국', '베트남', '홍콩', '태국'}
⑤ 핵심 포인트
  • 🚫 set은 중복 원소를 허용하지 않습니다. add('중국')은 이미 존재하므로 완전히 무시됩니다.
  • 🔗 update()는 다른 집합의 모든 원소를 한 번에 추가합니다. 중복인 '한국'은 자동으로 무시됩니다.
  • 🔀 set은 순서가 없습니다. print 결과의 원소 나열 순서는 실행마다 다를 수 있지만, 원소 자체는 동일합니다.
  • ⚠️ remove()는 없는 원소를 지우면 KeyError가 발생합니다. 안전하게 삭제하려면 discard()를 사용합니다.
