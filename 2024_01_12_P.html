Python 리스트 순회 + 문자열 인덱싱 누적 추적

Python · list · for loop · String Indexing · String Concatenation · Tracing
Q문제 코드
Python 
a = ["Seoul", "Kyeonggi", "Incheon", "Daejun", "Daegu", "Pusan"]
str = "S"

for i in a:
    str = str + i[1]    # 각 단어의 두 번째 문자(인덱스 1) 추가

print(str)

위 코드의 출력값은 무엇인가?

print() 출력 결과
Seynaau
"S" + e + y + n + a + a + u
각 도시명의 인덱스[1] 문자를 순서대로 누적
① 문제 유형

리스트의 각 문자열에서 인덱스 1(두 번째 문자)을 추출하여 초기 문자열에 순서대로 누적 연결(concatenation)하는 반복 추적 문제입니다.


  • 🔤i[1]은 문자열 i의 두 번째 문자(인덱스 1)를 의미합니다. 인덱스는 0부터 시작하므로 주의!
  • ➕str = str + i[1]은 기존 문자열 뒤에 새 문자를 이어 붙이는 연산입니다.
  • ⚠️함정: 초기값이 "S"이므로 결과는 7글자입니다. 리스트 길이(6) + 초기 1글자 = 7.

⑤ 핵심 주의 포인트
  • 🔤 Python 문자열 인덱스는 0부터 시작합니다. i[1]은 두 번째 문자입니다. i[0]이었다면 각 도시의 첫 글자(S, K, I, D, D, P)가 되었을 것입니다.
  • 📌 초기값 str = "S"로 이미 'S'가 포함되어 있으므로 최종 길이는 1 + 6 = 7글자입니다.
