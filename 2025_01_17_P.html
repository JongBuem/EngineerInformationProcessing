class Node:
    def __init__(self, value):
        self.value    = value
        self.children = []

def tree(li):
    nodes = [Node(i) for i in li]
    for i in range(1, len(li)):
        nodes[(i - 1) // 2].children.append(nodes[i])  # 힙 인덱스로 부모-자식 연결
    return nodes[0]

def calc(node, level=0):
    if node is None:
        return 0
    return (node.value if level % 2 == 1 else 0) \
         + sum(calc(n, level + 1) for n in node.children)

li   = [3, 5, 8, 12, 15, 18, 21]
root = tree(li)
print(calc(root))

위 코드의 출력값은 무엇인가?

print(calc(root)) 출력 결과
13
홀수 레벨(level=1)의 노드 값만 합산: 5 + 8 = 13
① 문제 유형

배열을 힙 인덱스 규칙으로 이진 트리에 배치한 뒤, 재귀로 홀수 레벨 노드 값만 합산하는 문제입니다.


  • 🔷힙 인덱스 부모 공식: 노드 i의 부모 = (i-1)//2 — 0-based 배열을 완전 이진 트리로 만드는 표준 방식입니다.
  • 🔷calc 규칙: level % 2 == 1(홀수)이면 node.value 합산, 짝수이면 0을 더합니다.
  • 🔷루트는 level=0(짝수)이므로 값을 더하지 않습니다. 자식 노드들만 레벨이 올라갑니다.
② tree() — 힙 인덱스로 이진 트리 생성

④ calc() 재귀 호출 전체 추적
호출 node.value level level%2 기여값 자식 합 반환
calc(12, 2) 12 2 짝수→0 0 자식 없음=0 0
calc(15, 2) 15 2 짝수→0 0 자식 없음=0 0

⑤ 레벨별 합산 규칙 요약
⑥ 핵심 주의 포인트
  • 🌳 힙 인덱스 공식 (i-1)//2: 1-based가 아닌 0-based 배열에서 완전 이진 트리를 만드는 표준 방법입니다.
  • 🔢 level % 2 == 1(홀수)일 때만 값을 더합니다. 루트는 level=0(짝수)이므로 값을 더하지 않습니다.
  • 🍃 리프 노드(level=2, 짝수)도 기여값이 0이고 자식도 없으므로 모두 0을 반환합니다.
  • 🖨️ 합산: level 1의 노드 값 5 + 8 = 13
