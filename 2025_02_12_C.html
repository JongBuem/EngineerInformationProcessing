C 원형 큐(Circular Queue) 연산 추적

C · Struct · Circular Queue · enq · deq · Modulo · Tracing
Q문제 코드
C 
#include <stdio.h>
#define SIZE 3

typedef struct {
    int a[SIZE];   // 크기 3인 배열
    int front;
    int rear;
} Queue;

void enq(Queue* q, int val) {
    q->a[q->rear] = val;
    q->rear = (q->rear + 1) % SIZE;   // 원형 증가
}

int deq(Queue* q) {
    int val = q->a[q->front];
    q->front = (q->front + 1) % SIZE;  // 원형 증가
    return val;
}

int main() {
    Queue q = {{0}, 0, 0};              // a={0,0,0}, front=0, rear=0
    enq(&q, 1); enq(&q, 2); deq(&q); enq(&q, 3);
    int first  = deq(&q);
    int second = deq(&q);
    printf("%d 그리고 %d", first, second);
    return 0;
}

위 코드의 출력값은 무엇인가?

printf 출력 결과
2 그리고 3
enq(1)→enq(2)→deq()=1→enq(3) 후 deq()=2, deq()=3
① 문제 유형

크기 3인 원형 큐(Circular Queue)의 enq/deq 연산 순서를 추적하여 최종 출력값을 구하는 문제입니다.


  • 🔷원형 큐 — rear와 front를 % SIZE로 증가시켜 배열 끝에서 처음으로 돌아옵니다.
  • 🔷enq — a[rear]에 값 저장 후 rear를 원형 증가
  • 🔷deq — a[front] 값 반환 후 front를 원형 증가 (FIFO — 먼저 넣은 것이 먼저 나옴)
② 원형 큐 구조 (SIZE=3)
③ 연산 단계별 상태 추적
단계 연산 a[0] a[1] a[2] front rear 반환값
초기 Queue q = {{0},0,0} 0 0 0 0 0 —
enq(1) a[0]=1, rear=(0+1)%3=1 1 0 0 0 1 —
enq(2) a[1]=2, rear=(1+1)%3=2 1 2 0 0 2 —
deq() val=a[0]=1, front=(0+1)%3=1 1 2 0 1 2 → 1 (버림)
enq(3) a[2]=3, rear=(2+1)%3=0 1 2 3 1 0 ← 원형! —
first=deq() val=a[1]=2, front=(1+1)%3=2 1 2 3 2 0 first = 2
second=deq() val=a[2]=3, front=(2+1)%3=0 1 2 3 0 0 second = 3
④ enq(3) 이후 큐 상태 — rear가 0으로 순환
⑤ 핵심 주의 포인트
  • 🔄 원형 큐의 핵심: (index + 1) % SIZE로 배열 끝에서 처음으로 순환합니다. enq(3) 후 rear = (2+1)%3 = 0
  • 📤 첫 번째 deq()는 반환값(1)을 버립니다. 이후 front=1이 됩니다.
  • 📌 enq(3) 시 a[2]=3, rear가 0으로 순환 — 배열에는 {1, 2, 3}이지만 실제 큐 내용은 front=1부터: 2 → 3
  • 🖨️ 최종: first=2, second=3 → 출력 "2 그리고 3"
