Java String.split() 분리 결과 인덱스 추적

Java · String · split() · Array Index · Tracing
Q문제 코드
Java 
public class Main {
    public static void main(String[] args) {
        String str = "ITISTESTSTRING";
        String[] result = str.split("T");   // "T" 기준으로 분리
        System.out.print(result[3]);        // 인덱스 3 출력
    }
}

위 코드의 출력값은 무엇인가?

System.out.print() 출력 결과
S
"ITISTESTSTRING".split("T") → result[3] = "S"
① 문제 유형

String.split()의 구분자 기준 분리 결과 배열을 정확히 파악하고, 특정 인덱스 원소를 읽는 문제입니다.


  • ✂️str.split("T")는 문자열을 "T"가 나타나는 위치마다 잘라 String 배열로 반환합니다. "T" 자체는 결과에 포함되지 않습니다.
  • ⚠️함정: "T"가 연속/인접 위치에 있으면 빈 문자열""이 원소로 들어갑니다. 이 문제에서 "T" 사이 간격을 정확히 세는 것이 핵심입니다.

⑤ String.split() 동작 규칙
규칙 설명 이 문제 적용
구분자 제외 분리 기준 문자는 결과에 포함되지 않음 "T"는 결과 배열에 없음
앞쪽 빈 문자열 유지 문자열이 구분자로 시작하면 result[0]="" "I"로 시작하므로 해당 없음
뒤쪽 빈 문자열 제거 trailing empty string은 기본 제거 끝이 "G"이므로 해당 없음
구분자 n개 → 배열 크기 n+1 "T" 4개 → 결과 배열 크기 5 result.length = 5
result[3] 4번째 T(idx 9)와 3번째 T(idx 7) 사이 문자 idx 8 = "S"
⑥ 핵심 주의 포인트
  • ✂️ 구분자 "T" 등장 횟수 4번 → 분리된 조각 5개. 배열 인덱스 0~4.
  • 🔢 분리 결과: ["I", "IS", "ES", "S", "RING"]
    — T(idx1) 앞: "I" / T(idx4) 앞: "IS" / T(idx7) 앞: "ES" / T(idx9) 앞: "S" / 나머지: "RING"
  • ⚠️ 만약 구분자가 연속으로 나오면 (예: "TT") 사이에 빈 문자열 ""이 삽입됩니다. 이 문제에는 해당 없음.
