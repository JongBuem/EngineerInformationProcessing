____빈칸____ 에 들어갈 키워드는 무엇인가?

빈칸에 들어갈 키워드
implements
class WashingMachine implements Machine { ... }
① 문제 유형

Java에서 클래스가 인터페이스를 구현할 때 사용하는 키워드를 묻는 빈칸 채우기 문제입니다.
인터페이스는 메서드의 명세(계약)만 정의하고, 구현 클래스가 실제 동작을 작성합니다.

② 정답 적용 코드
Java — 정답 적용 
class WashingMachine implements Machine {
    // Machine 인터페이스의 run()을 반드시 구현해야 한다
    public void run() {
        System.out.println("Washing machine running");
    }
}
③ 인터페이스 구현 구조 (UML)
④ extends vs implements — 비교
키워드 사용 대상 관계 다중 적용 예시
implements 클래스 → 인터페이스 구현 (계약 이행) 가능 (다중) class A implements B, C
extends 클래스 → 클래스
인터페이스 → 인터페이스 		상속 단일 (클래스 기준) class A extends B
⑤ 핵심 정리
  • 📌클래스가 인터페이스를 구현할 때는 반드시 implements 키워드를 사용합니다.
  • 📋인터페이스의 모든 추상 메서드를 구현하지 않으면 컴파일 오류가 발생합니다.
  • 🔢implements는 여러 인터페이스를 동시에 구현할 수 있습니다. 예: class A implements B, C, D
  • 🖨️실행 결과: "Washing machine running"
