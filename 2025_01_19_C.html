C 구조체 배열 & 비트 AND 마스킹 합산 추적

C · Struct · Bitwise AND · Hex · Masking · Array · Tracing
Q문제 코드
C 
#include <stdio.h>

typedef struct student {
    char* name;
    int   score[3];
} Student;

int dec(int enc) {
    return enc & 0xA5;        // 마스크: 1010 0101
}

int sum(Student* p) {
    return dec(p->score[0])
         + dec(p->score[1])
         + dec(p->score[2]);
}

int main() {
    Student s[2] = {
        "Kim", {0xA0, 0xA5, 0xDB},
        "Lee", {0xA0, 0xED, 0x81}
    };
    int result = 0;
    for (int i = 0; i < 2; i++)
        result += sum(&s[i]);
    printf("%d", result);
    return 0;
}

① 문제 유형

구조체 배열의 각 멤버에 비트 AND 마스킹을 적용하여 합산하는 16진수 비트 연산 추적 문제입니다.


  • 🔷dec() 함수: enc & 0xA5 — 마스크 0xA5 = 1010 0101과 AND 연산합니다. 마스크가 0인 비트는 항상 0, 1인 비트는 원래 값을 유지합니다.
  • 🔷sum() 함수: 구조체 포인터로 3개의 score에 dec()를 적용하여 합산합니다.
  • 🟡16진수를 2진수로 변환한 뒤 비트별 AND 연산을 정확히 수행해야 합니다.
② 마스크 0xA5 = 1010 0101 이해
③ dec() 비트 AND 연산 전체 계산

dec(0xA0) — Kim·Lee 공통 score[0]

dec(0xA5) — Kim score[1]

dec(0xDB) — Kim score[2]

dec(0xED) — Lee score[1]

dec(0x81) — Lee score[2]

④ sum() 결과 및 최종 합산
학생 score[0] dec() score[1] dec() score[2] dec() sum 합계
Kim 0xA0 160 0xA5 165 0xDB 129 160+165+129 = 454
Lee 0xA0 160 0xED 165 0x81 129 160+165+129 = 454
result = 454 + 454 = 908

※ 두 학생의 dec() 결과가 같은 것은 우연이 아닙니다. score[0]은 동일 값이고,
  0xA5&0xA5=0xA5, 0xED&0xA5=0xA5, 0xDB&0xA5=0x81, 0x81&0xA5=0x81 이 일치합니다.

⑤ 핵심 주의 포인트
  • 🔢 16진수 → 2진수 변환이 필수입니다. 16진수 한 자리 = 4비트이므로 0xDB = 0xD(1101) 0xB(1011) = 1101 1011.
  • 🔧 AND 연산 규칙: 1&1=1, 1&0=0, 0&1=0, 0&0=0 — 마스크 비트가 0인 자리는 무조건 0이 됩니다.
  • ⚡ dec(0xA5)=165, dec(0xED)=165 — 다른 값이지만 마스크 후 같은 결과가 나오는 것이 핵심 함정입니다.
  • 🖨️ 최종 출력: 908
