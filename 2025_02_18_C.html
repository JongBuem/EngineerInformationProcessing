C 연결 리스트 스택 — 문자열 역순 출력 추적

C · Struct · Linked List · Stack · Head Insert · String Reverse · Tracing
Q문제 코드
C 
#include <stdio.h>
#include <stdlib.h>

struct node {
    char c;
    struct node* p;
};

struct node* func(char* s) {
    struct node* h = NULL, *n;
    while (*s) {
        n = malloc(sizeof(struct node));
        n->c = *s++;           // 현재 문자 저장 후 s 전진
        n->p = h;              // 새 노드 다음 = 기존 헤드 (앞에 삽입)
        h = n;                 // 헤드 갱신
    }
    return h;
}

int main() {
    struct node* n = func("BEST");
    while (n) {
        putchar(n->c);
        struct node* t = n;
        n = n->p;
        free(t);
    }
    return 0;
}

위 코드의 출력값은 무엇인가?

putchar 출력 결과
TSEB
"BEST" 헤드 삽입(스택) → T→S→E→B→NULL 순 출력
① 문제 유형

연결 리스트에 문자를 헤드 삽입(head insert)하여 스택처럼 역순으로 쌓은 뒤 출력하는 문제입니다.


  • 🔷헤드 삽입 — n->p = h; h = n; 패턴은 새 노드를 항상 리스트 앞에 붙입니다. 결과적으로 입력 순서의 역순이 됩니다.
  • 🔷이 구조는 스택(LIFO)과 동일합니다 — 마지막에 넣은 'T'가 헤드가 됩니다.
  • 🔷*s++ — 후위 증가: 현재 *s 값을 먼저 읽고, 그 다음 포인터를 전진시킵니다.
② func("BEST") 헤드 삽입 과정
③ func 루프 단계별 추적
반복 읽은 문자 n->c n->p h (헤드) 상태
초기 — — — NULL
1 'B' 'B' NULL B → NULL
2 'E' 'E' → B E → B → NULL
3 'S' 'S' → E S → E → B → NULL
4 'T' 'T' → S T → S → E → B → NULL
종료 '\0' 루프 종료 return h (= T 노드)
④ main — putchar 출력 순서
순서 n 현재 노드 putchar 출력 n = n->p (다음)
1 T 노드 T → S 노드
2 S 노드 S → E 노드
3 E 노드 E → B 노드
4 B 노드 B → NULL
종료 n == NULL → while 종료
⑤ 핵심 주의 포인트
  • 🔄 헤드 삽입 = 스택(LIFO). n->p = h; h = n; 패턴으로 새 노드가 항상 맨 앞에 붙으므로 "BEST" 입력 → T→S→E→B 순서로 연결됩니다.
  • 📝 *s++ — 후위 증가: *s를 먼저 읽은 뒤 포인터 s를 1 증가시킵니다. ++*s(값 증가)와 혼동 금지!
  • 🗑️ free(t) — 출력 후 이미 읽은 노드를 즉시 해제합니다. 메모리 누수 없는 올바른 패턴입니다.
  • 🖨️ "BEST" 입력 → 역순 → TSEB 출력
