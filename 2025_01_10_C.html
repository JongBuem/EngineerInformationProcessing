C 전역 배열 문자 삽입 정렬 추적

C · Global Array · Char ASCII · Insertion · Shift · printf · Tracing
Q문제 코드
C 
#include <stdio.h>
char Data[5] = {'B', 'A', 'D', 'E'};  // Data[4] = '\0' (자동)
char c;

int main() {
    int i, temp, temp2;
    c = 'C';

    printf("%d\n", Data[3] - Data[1]);  // 'E' - 'A' = ?

    /* ① 삽입 위치 탐색 */
    for (i = 0; i < 5; ++i) {
        if (Data[i] > c) break;
    }

    /* ② c 삽입, 기존 원소 오른쪽 이동 */
    temp = Data[i];
    Data[i] = c;
    i++;
    for (; i < 5; ++i) {
        temp2  = Data[i];
        Data[i] = temp;
        temp   = temp2;
    }

    /* ③ 결과 출력 */
    for (i = 0; i < 5; i++)
        printf("%c", Data[i]);
}

① 문제 유형

정렬된 배열에 새 문자를 삽입하고 뒤 원소들을 오른쪽으로 이동(시프트)하는 삽입 정렬 응용 문제입니다.


  • 🔷전역 배열 초기화: char Data[5] = {'B','A','D','E'} — 원소가 4개이므로 Data[4] = '\0'(0)으로 자동 초기화됩니다.
  • 🔷ASCII 연산: Data[3] - Data[1] = 'E'(69) - 'A'(65) = 4
  • 🔷삽입 알고리즘: 첫 번째 루프로 삽입 위치를 찾고, 두 번째 루프로 뒤 원소들을 한 칸씩 오른쪽으로 밀어냅니다.

⑦ 핵심 주의 포인트
  • ⚠️ 전역 배열 부분 초기화: char Data[5] = {'B','A','D','E'} — 4개만 명시하면 Data[4]는 '\0'(0)으로 자동 초기화됩니다. 이 빈 슬롯 덕분에 삽입이 가능합니다.
  • 🔍 탐색 루프는 첫 번째로 c보다 큰 원소를 찾아 break합니다. 'D'(68) > 'C'(67)이므로 i=2에서 중단.
  • 🔄 시프트 루프는 temp를 버퍼로 사용해 D→[3], E→[4]로 연쇄 이동합니다.
  • 🖨️ 출력 순서: 1행 "4" → 2행 "BACDE"
