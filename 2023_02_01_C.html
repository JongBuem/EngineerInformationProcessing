[조건]
입력값이  5 4 3 2 1  일 경우
출력값이  4 3 2 1 5  로 출력되어야 한다.

빈칸에 들어갈 식은 무엇인가?

빈칸에 들어갈 식
n[(i+1)%5]
i=0 → n[1]=4  |  i=1 → n[2]=3  |  i=2 → n[3]=2
i=3 → n[4]=1  |  i=4 → n[0]=5 (순환!)
① 문제 유형

배열을 한 칸씩 앞으로 순환 이동(Circular Left Shift)하여 출력하는 식을 완성하는 빈칸 채우기 문제입니다.


  • 📌입력 5 4 3 2 1 → n[0]=5, n[1]=4, n[2]=3, n[3]=2, n[4]=1
  • 🎯출력 4 3 2 1 5 → 첫 번째 원소(5)가 맨 뒤로 가는 왼쪽 순환

③ 빈칸 식 도출 과정
i=0 일 때 n[1] 출력 → 인덱스 = 1 = (0+1)%5
i=1 일 때 n[2] 출력 → 인덱스 = 2 = (1+1)%5
i=2 일 때 n[3] 출력 → 인덱스 = 3 = (2+1)%5
i=3 일 때 n[4] 출력 → 인덱스 = 4 = (3+1)%5
i=4 일 때 n[0] 출력 → 인덱스 = 0 = (4+1)%5 ← 순환!

∴ n[(i+1) % 5]
④ 단계별 실행 추적표
i (i+1)%5 접근 인덱스 n[인덱스] 출력
0 (0+1)%5 = 1 n[1] 4 4
1 (1+1)%5 = 2 n[2] 3 3
2 (2+1)%5 = 3 n[3] 2 2
3 (3+1)%5 = 4 n[4] 1 1
4 (4+1)%5 = 0 n[0] 5 5 ← 순환!
⑤ 핵심 포인트
  • 🔁 나머지 연산(%)이 순환의 핵심입니다. i=4일 때 (4+1)%5=0 이 되어 첫 번째 원소로 돌아옵니다.
  • 📐 일반화: 크기 N인 배열에서 한 칸 왼쪽 순환 출력 → n[(i+1) % N]
  • ✅ 정답: n[(i+1)%5] → 출력 결과: 43215
