C 삼항 연산자 + 좌측 시프트(<<) 조건 분기 추적

C · Ternary Operator · Left Shift (<<) · if-else · Bitwise · Tracing
Q문제 코드
C 
int main() {

    int v1 = 0, v2 = 35, v3 = 29;

    if (v1 > v2 ? v2 : v1) {   // 삼항 연산자가 if 조건
        v2 = v2 << 2;
    } else {
        v3 = v3 << 2;
    }

    printf("%d", v2 + v3);
}

위 코드의 출력값은 무엇인가?

printf() 출력 결과
151
v2=35 (변경 없음) + v3=29<<2=116 → 35+116 = 151
① 문제 유형

삼항 연산자의 결과가 if 조건으로 사용되고, 선택된 분기에서 좌측 비트 시프트(<<)를 적용하는 이중 함정 문제입니다.


  • ⚠️함정 1: v1 > v2 ? v2 : v1의 결과가 if 조건값입니다. v1=0, v2=35이므로 0 > 35는 false → 삼항 결과 = v1 = 0
  • ⚠️함정 2: if(0)는 false → else 분기 실행. v2가 아닌 v3이 시프트됩니다.
  • 🔷n << 2는 n을 2비트 왼쪽으로 이동 = n × 4. 29 << 2 = 116
