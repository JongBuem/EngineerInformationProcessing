Java 메서드 오버로딩 + 재귀 호출 추적

Java · Method Overloading · Recursion · String vs int · Dispatch · Tracing
Q문제 코드
Java 
public class Main {
    public static void main(String[] args) {
        System.out.println(calc("5"));   // String 인수 → calc(String) 호출
    }

    /* ① calc(int) — int 인수 버전: value-1, value-2 피보나치 */
    static int calc(int value) {
        if (value <= 1) return value;
        return calc(value - 1) + calc(value - 2);
    }

    /* ② calc(String) — String 인수 버전: value-1, value-3 */
    static int calc(String str) {
        int value = Integer.valueOf(str);
        if (value <= 1) return value;
        return calc(value - 1) + calc(value - 3);   // int 연산 → calc(int) 호출!
    }
}

위 코드의 출력값은 무엇인가?

System.out.println 출력 결과
4
calc("5") → calc(int 4) + calc(int 2) → 3 + 1 = 4
① 문제 유형

메서드 오버로딩(Overloading)으로 인해 String 버전과 int 버전 중 어떤 calc()가 호출되는지 파악하는 것이 핵심인 재귀 추적 문제입니다.


  • ⚠️핵심 함정: calc(String) 안에서 calc(value-1)을 호출할 때, value-1은 int 연산 결과이므로 calc(int) 버전이 호출됩니다. String 버전은 최초 한 번만 실행됩니다.
  • 🔵두 버전의 점화식이 다릅니다: calc(int)는 f(n-1)+f(n-2) (피보나치), calc(String)은 f(n-1)+f(n-3)입니다.
② 두 calc() 오버로드 비교
calc(String str) — 단 1회만 호출
인수 타입: String
점화식: f(n-1) + f(n-3)
내부에서 value-1, value-3은
int이므로 calc(int) 호출!
calc(int value) — 이후 모든 재귀에 사용
인수 타입: int
점화식: f(n-1) + f(n-2)
기저: value ≤ 1 → return value
(0→0, 1→1)
③ calc(String "5") 진입 — int 재귀로 전환되는 순간

calc("5") 호출
  → value = Integer.valueOf("5") = 5
  → 5 > 1 이므로:
  → return calc(5-1) + calc(5-3)
  → return calc(int 4) + calc(int 2)   ← 이제부터 calc(int)만 호출!

④ 재귀 호출 트리 (calc(int) 버전 — 피보나치)
⑤ 단계별 반환값 추적표
호출 타입 조건 왼쪽 재귀 오른쪽 재귀 반환값
calc(int 0) int 0 ≤ 1 — — 0 (기저)
calc(int 1) int 1 ≤ 1 — — 1 (기저)
calc(int 2) int 2 > 1 calc(1)=1 calc(0)=0 1+0 = 1
calc(int 3) int 3 > 1 calc(2)=1 calc(1)=1 1+1 = 2
calc(int 4) int 4 > 1 calc(3)=2 calc(2)=1 2+1 = 3
calc(int 2) int 2 > 1 calc(1)=1 calc(0)=0 1+0 = 1
calc(String "5") String 5 > 1 calc(4)=3 calc(2)=1 3+1 = 4
⑥ 핵심 주의 포인트
  • ⚠️ 오버로딩 디스패치는 컴파일 타임 타입 기준: calc(String) 안에서 calc(value-1)을 호출할 때 value-1은 int 연산이므로 calc(int)가 호출됩니다. String 버전은 최초 한 번만 실행됩니다.
  • 🔢 calc(String)의 점화식은 f(n-1) + f(n-3)이지만, 실제로 재귀는 calc(int)의 f(n-1)+f(n-2)로 수행됩니다.
  • 🔁 최종 계산: calc("5") = calc(int 4) + calc(int 2) = 3 + 1 = 4
