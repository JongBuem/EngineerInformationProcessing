C 문자 분류 + 시프트 인코딩 변환 추적

C · isupper · islower · isdigit · ASCII · Modulo Shift · ctype.h · Tracing
Q문제 코드
C 
#include <stdio.h>
#include <ctype.h>

int main() {
    char*p = "It is 8";
    char result[100];
    int i;

    for (i = 0; p[i] != '\0'; i++) {
        if (isupper(p[i]))
            result[i] = (p[i] - 'A' + 5) % 26 + 'A';  // 대문자: +5 순환
        else if (islower(p[i]))
            result[i] = (p[i] - 'a' + 10) % 26 + 'a'; // 소문자: +10 순환
        else if (isdigit(p[i]))
            result[i] = (p[i] - '0' + 3) % 10 + '0';  // 숫자: +3 순환
        else
            result[i] = p[i];                          // 그 외: 그대로
    }

    result[i] = '\0';
    printf("%s\n", result);
    return 0;
}

① 문제 유형

각 문자의 종류(대문자/소문자/숫자/기타)에 따라 서로 다른 모듈러(%) 시프트 인코딩을 적용하는 문자 변환 추적 문제입니다.


  • 🔵대문자: 알파벳 위치에 +5를 더한 뒤 26으로 나머지 → 26자 순환
  • 🟢소문자: 알파벳 위치에 +10을 더한 뒤 26으로 나머지 → 26자 순환
  • 🟡숫자: 숫자 값에 +3을 더한 뒤 10으로 나머지 → 0~9 순환
  • ⬜공백/기타: 변환 없이 그대로 복사

⑥ 핵심 주의 포인트
  • 🔢 모듈러 순환 시프트: (위치 + 이동량) % 범위 공식으로 알파벳/숫자 범위를 벗어나지 않고 순환됩니다. 's'(18)+10=28 → 28%26=2 → 'c'
  • 📌 대문자 +5, 소문자 +10, 숫자 +3 — 이동량이 각각 다름에 주의합니다.
  • ⬜ 공백은 isupper, islower, isdigit 모두 해당 안 되므로 else 분기에서 그대로 복사됩니다.
  • 🖨️ 최종 출력: Nd sc 1
