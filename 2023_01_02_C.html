C 문자 배열 & 포인터 출력 추적

C · char array · char* pointer · %s · %c · *p · *a · for loop
Q문제 코드
C 
#include <stdio.h>

int main() {
    char a[] = "Art";     // a[0]='A', a[1]='r', a[2]='t', a[3]='\0'
    char* p = NULL;
    p = a;               // p가 a[0]의 주소를 가리킴

    printf("%s\n", a);   // ① 배열명 → 문자열 전체
    printf("%c\n", *p);  // ② p가 가리키는 첫 문자
    printf("%c\n", *a);  // ③ 배열명 역참조 → 첫 문자
    printf("%s\n", p);   // ④ 포인터 → 문자열 전체

    for(int i = 0; a[i] != '\0'; i++)
        printf("%c", a[i]); // ⑤ 문자 하나씩, 줄바꿈 없음
}

① 문제 유형

C 문자 배열과 char* 포인터가 동일한 주소를 가리킬 때 %s/%c 서식과 역참조(*) 연산의 출력 결과를 추적하는 문제입니다.

⑤ 핵심 포인트
  • 📌 배열명 a는 첫 번째 원소의 주소입니다. p = a 이후 p와 a는 같은 주소를 가리킵니다.
  • 🔍 *p와 *a는 모두 역참조 → a[0] = 'A'를 반환합니다.
  • 📋 %s는 '\0'을 만날 때까지 문자열 전체를, %c는 문자 단 하나를 출력합니다.
  • 🔁 for 루프의 printf("%c", a[i])는 \n 없이 한 글자씩 출력 → 마지막 줄은 줄바꿈 없음.
