C 배열 스택 — push/pop while 루프 출력 추적 (수정본)

C · Stack · Array · push(into) · pop(take) · Empty 감지 · Tracing
Q문제 코드
C 
#include <stdio.h>
#define MAX_SIZE 10

int isWhat[MAX_SIZE];
int point = -1;

void into(int num) {
    if (point >= 10) printf("Full");
    isWhat[++point] = num;
}

int take() {
    if (isempty() == 1) printf("Empty");
    return isWhat[point--];
}

int isempty() {
    if (point == -1) return 1;
    return 0;
}

int main(int argc, char const *argv[]){
    int e;
    into(5); into(2);
    while(!isempty()){
        printf("%d", take());  
        into(4); into(1); printf("%d", take());
        into(3); printf("%d", take()); printf("%d", take());
        into(6); printf("%d", take()); printf("%d", take()); printf("%d", take());
    return 0;
}

⑥ 핵심 포인트
  • ✨ 새로 추가된 printf("%d", take())가 루프 첫 줄에서 초기 top 값 2를 바로 꺼냅니다. 이후 스택에는 [5]만 남아 흐름이 달라집니다.
  • ⚠️ 마지막 줄 take()×3 수행 시 6, 5를 꺼낸 후 스택이 비어있는 상태에서 한 번 더 take() 호출 → isempty()==1 → "Empty" 출력
  • 🔄 while 루프는 1회만 수행됩니다. 마지막 take() 후 point=-1이 되어 다음 조건 검사에서 종료됩니다.
