#include <stdio.h>
int main() {
    int E[] = {64, 25, 12, 22, 11};
    int n = sizeof(E) / sizeof(E[0]);  // n = 5
    int i = 0;
    do {
        int j = i + 1;
        do {
            if (E[i] ( 　?　 ) E[j]) {   // ★ 빈칸
                int tmp = E[i];
                E[i] = E[j];
                E[j] = tmp;
            }
            j++;
        } while (j < n);
        i++;
    } while (i < n - 1);
    for(int i = 0; i < n; i++)
        printf("%d ", E[i]);
}

빈칸에 들어갈 연산자는 무엇인가?

빈칸에 들어갈 연산자
>
if (E[i] > E[j])
E[i]가 E[j]보다 크면 교환 → 오름차순 정렬
최종 출력: 11 12 22 25 64
① 문제 유형

do-while 이중 루프로 구현된 선택 정렬(Selection Sort)에서 swap 조건의 비교 연산자를 찾는 빈칸 채우기 문제입니다.

② 알고리즘 구조 분석
외부 루프 i: 0 ~ n-2 (기준 원소)
내부 루프 j: i+1 ~ n-1 (비교 대상)
swap 조건: E[i] ? E[j] 이면 E[i]↔E[j] 교환

결과 배열이 오름차순이 되려면?
→ E[i]가 E[j]보다 크면(>) 교환 = 작은 값을 앞으로
③ 연산자 선택 이유
연산자 의미 정렬 결과 선택?
> E[i]가 더 크면 swap ✅ 오름차순 (작은 값이 앞으로) ✓ 정답
< E[i]가 더 작으면 swap ❌ 내림차순 오답
>= 크거나 같으면 swap 불필요한 swap 발생 오답
== 같을 때만 swap 정렬 불가 오답

  • 🔄 이 코드는 선택 정렬(Selection Sort) 입니다. 기준 원소 E[i]와 이후 모든 원소 E[j]를 비교해 교환합니다.
