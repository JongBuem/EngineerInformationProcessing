C 연산자 우선순위 & 논리 연산 추적

C · Operator Precedence · Relational · Logical AND · Tracing
Q문제 코드
C 
#include <stdio.h>
int main() {
    int x=7, y=4, z;

    z = y%3 > 1;          // ① 나머지 → 비교 → z에 저장
    z = x>5 && z;         // ② 비교 → 논리AND → z에 저장

    return z;             // z의 최종값을 반환
}

프로그램이 종료될 때 z의 최종값(return z)은 무엇인가?

return z 의 최종값
0
y%3=1 > 1 → false(0) │ x>5 && 0 → 0
① 문제 유형

연산자 우선순위에 따라 식을 올바르게 분해하고, 논리 연산 결과를 추적하는 문제입니다.


  • 🔷연산자 우선순위 — %(나머지) > >(비교) > &&(논리AND) 순서로 평가됩니다.
  • 🔷C의 불리언 — C에서 비교/논리 연산 결과는 참이면 1, 거짓이면 0 (정수)으로 저장됩니다.
  • 🔷&& 단락 평가 — 왼쪽이 0(거짓)이면 오른쪽을 평가하지 않고 즉시 0을 반환합니다.
② 이 코드에서 사용된 연산자 우선순위
우선순위 연산자 이름 결합 방향
높음 ① % 나머지(모듈로) 좌 → 우
② > 크다 (관계 연산자) 좌 → 우
낮음 ③ && 논리 AND 좌 → 우

⑤ 핵심 주의 포인트
  • ⚖️ y%3 > 1은 (y%3) > 1로 파싱됩니다. %가 >보다 우선순위가 높습니다.
  • 🔢 4 % 3 = 1, 1 > 1은 거짓(0)입니다. '크다'이지 '이상'이 아님에 주의하세요.
  • ⚡ &&(논리 AND)는 왼쪽이 1이더라도 오른쪽 z=0이 있으므로 결과는 0입니다.
  • 📝 이 코드에는 printf가 없습니다. return z는 OS에 종료 코드 0을 반환하며, 화면 출력은 없습니다.
