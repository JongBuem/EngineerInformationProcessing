C 이중 for 루프 — 공통 문자 찾기 추적

C · nested for loop · char* · string intersection · Tracing
Q문제 코드
C 
#include <stdio.h>

int main() {
    char* a = "qwer";    // 길이 4
    char* b = "qwtety";  // 길이 6

    for(int i = 0; a[i] != '\0'; i++) {       // 외부: a 순회
        for(int j = 0; b[j] != '\0'; j++) {   // 내부: b 순회
            if(a[i] == b[j]) printf("%c", a[i]); // 일치하면 출력
        }
    }
}

④ 단계별 추적표
i a[i] b에서 찾은 위치 결과 출력
0 'q' b[0]='q' 일치 ✓ HIT q

2 'e' b[3]='e' 일치 ✓ HIT e
3 'r' b 전체 순회 — 없음 ✗ MISS —
⑤ 핵심 포인트
  • 🔄 외부 루프가 a의 순서를 결정합니다. a = "qwer" 순서로 탐색하므로 출력도 q → w → e 순입니다.
  • 🔍 내부 루프는 매 i마다 b 전체를 다시 순회합니다. b = "qwtety" 에서 'w'는 b[1]에 존재합니다.
  • ❌ 'r'은 b = "qwtety" 에 존재하지 않아 아무것도 출력되지 않습니다.
  • 📋 printf("%c", a[i])는 \n 없이 출력 → 결과는 한 줄에 qwe.
