Python 슬라이딩 윈도우 부분문자열 카운팅 + f-string

Python · String · Sliding Window · Substring Count · f-string · Tracing
Q문제 코드
Python 
def cnt(str, p):
    result = 0
    for i in range(len(str)):            # i = 0 ~ 10
        sub = str[i : i+len(p)]          # 길이 len(p)의 슬라이딩 윈도우
        if sub == p:
            result += 1
    return result

str = "abdcabcabca"   # 길이 11
p1  = "ca"
p2  = "ab"

print(f'ab{cnt(str, p1)} ca{cnt(str, p2)}')

위 코드의 출력값은 무엇인가?

print() 출력 결과
ab3 ca3
cnt("abdcabcabca","ca")=3 · cnt("abdcabcabca","ab")=3
① 문제 유형

슬라이딩 윈도우(길이 2) 방식으로 부분 문자열 출현 횟수를 세고, f-string으로 결합 출력하는 문제입니다.


  • ⚠️핵심 함정: f-string에서 'ab{cnt(str,p1)} ca{cnt(str,p2)}'의 ab와 ca는 리터럴 문자열이고 패턴 p1, p2가 아닙니다. 출력에 그대로 포함됩니다.
  • 🔷str[i:i+len(p)] — 슬라이딩 윈도우가 한 칸씩 이동하며 길이 2의 부분 문자열을 패턴과 비교합니다.

⑤ f-string 조합 분석
f'ab{cnt(str, p1)} ca{cnt(str, p2)}'
= f'ab3 ca3'
= "ab3 ca3"
'ab' → 리터럴 문자열 (패턴 p2 "ab"와 무관)
{cnt(str, p1)} → cnt(str, "ca") = 3
' ca' → 리터럴 문자열 (공백 포함, 패턴 p1 "ca"와 무관)
{cnt(str, p2)} → cnt(str, "ab") = 3
⑥ 핵심 주의 포인트
  • 🔍 슬라이딩 윈도우: str[i:i+len(p)]는 i=0부터 마지막까지 이동하며 길이 len(p)의 부분 문자열을 추출합니다. i=10에서는 한 글자만 남아 "ca"/"ab"와 불일치합니다.
  • ⚠️ f-string 함정: f'ab{...} ca{...}'에서 ab와 ca는 리터럴이고, {} 안의 함수 호출 결과가 삽입됩니다. 패턴 이름과 헷갈리지 않도록 주의합니다.
  • 📊 "abdcabcabca"에서 "ca" 3회 (인덱스 3, 6, 9), "ab" 3회 (인덱스 0, 4, 7)
  • 🖨️ 최종 출력: "ab3 ca3"
