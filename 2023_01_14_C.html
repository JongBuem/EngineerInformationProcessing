#include <stdio.h>

void swap(int *a, int idx1, int idx2) {
    int t = a[idx1];
    a[idx1] = a[idx2];
    a[(① ?)] = t;  // ★ 빈칸 ①
}

void Usort(int *a, int len) {
    for(int i = 0; i < len - 1; i++)
        for(int j = 0; j < len - 1 - i; j++)
            if(a[j] > a[j+1])
                swap(a, j, j+1);
}

void main() {
    int a[] = {85, 75, 50, 100, 95};
    int nx = 5;
    Usort(a, (② ?));  // ★ 빈칸 ②
}

① 문제 유형

포인터 기반 swap 함수와 버블 정렬(Bubble Sort) 호출에서 빈칸을 완성하는 문제입니다. swap 로직의 3단계 교환 패턴과 함수에 올바른 인자를 전달하는 방법을 이해해야 합니다.

② 빈칸 ① 분석 — swap 완성
swap의 3단계:
① t = a[idx1]    ← idx1의 값을 임시 저장
② a[idx1] = a[idx2] ← idx2의 값을 idx1에 복사
③ a[idx2] = t     ← 임시값(t)을 idx2에 저장

∴ 빈칸 ① = idx2

⑤ 핵심 포인트
  • 🔄 swap 3단계: t=a[idx1] → a[idx1]=a[idx2] → a[idx2]=t. 마지막에 반드시 idx2에 t를 넣어야 교환 완성.
  • 📏 Usort의 두 번째 매개변수 len은 배열의 길이입니다. nx=5가 그 값을 담고 있습니다.
  • 🫧 이 알고리즘은 버블 정렬(Bubble Sort)입니다. 인접한 두 원소를 비교해 큰 값을 오른쪽으로 밀어 오름차순 정렬합니다.
  • 📋 최종 결과: 50 75 85 95 100 (오름차순)
