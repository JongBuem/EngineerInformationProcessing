2list — 순서 있는 가변 컬렉션

가장 많이 사용하는 자료구조. 순서 있고, 중복 허용, 수정 가능합니다. 인덱스는 0부터, 음수 인덱스로 뒤에서 접근 가능합니다.

Python 
a = [10, 20, 30, 40, 50]

# 인덱싱
a[0]    # 10  (앞에서 첫 번째)
a[-1]   # 50  (뒤에서 첫 번째)
a[-2]   # 40  (뒤에서 두 번째)

# 주요 메서드
a.append(60)      # 맨 뒤에 추가 → [10,20,30,40,50,60]
a.insert(1, 99)   # 인덱스 1에 삽입 → [10,99,20,30,40,50]
a.remove(30)      # 값 30 제거 (처음 발견된 것)
a.pop()           # 맨 뒤 제거 & 반환
a.pop(0)          # 인덱스 0 제거 & 반환
a.reverse()       # 제자리 역순 (반환값 None)
a.sort()           # 제자리 정렬 (반환값 None)
len(a)             # 길이
30 in a             # True/False

⚠ 함정: a.reverse(), a.sort()는 제자리 변경이며 반환값은 None입니다. b = a.reverse() 하면 b = None!

3dict — 키-값 쌍 자료구조

키(key)와 값(value)의 쌍으로 데이터를 저장합니다. 키는 중복 불가, 같은 키에 새 값을 대입하면 덮어씁니다.

Python 
d = {'name': 'Alice', 'age': 20, 'score': 95}

# 접근
d['name']          # 'Alice'
d.get('age')       # 20  (없으면 None, 에러 없음)
d.get('x', 0)      # 0   (기본값 지정)

# 추가/수정/삭제
d['grade'] = 'A'    # 추가
d['age'] = 21       # 수정 (덮어쓰기)
del d['score']      # 삭제

# 순회
for k in d.keys():   # 키만
for v in d.values(): # 값만
for k, v in d.items(): # 키+값 쌍

# 딕셔너리 컴프리헨션
squares = {x: x*x for x in range(5)} # {0:0, 1:1, 2:4, 3:9, 4:16}
4set — 순서 없는 중복 불가 집합

set은 중복 원소를 자동으로 제거하고 순서가 없습니다. 수학의 집합 연산(합집합, 교집합, 차집합)을 지원합니다.

Python 
s = {1, 2, 3, 2, 1}   # → {1, 2, 3} 중복 자동 제거

# 원소 추가/제거
s.add(4)         # 추가. 이미 있으면 무시
s.remove(2)     # 제거. 없으면 KeyError
s.discard(9)    # 제거. 없어도 에러 없음 (안전)
s.update({5,6}) # 여러 원소 한 번에 추가

# 집합 연산
a = {1,2,3}; b = {2,3,4}
a | b   # {1,2,3,4}  합집합  (a.union(b))
a & b   # {2,3}      교집합  (a.intersection(b))
a - b   # {1}        차집합  (a.difference(b))
a ^ b   # {1,4}      대칭 차집합

⚠ 주의: s = {}는 빈 set이 아니라 빈 dict입니다! 빈 set은 s = set()으로 만들어야 합니다.

5tuple — 불변 시퀀스

tuple은 list와 거의 같지만 값을 변경할 수 없습니다(immutable). 함수에서 여러 값을 반환하거나, 변경되어선 안 되는 데이터를 다룰 때 사용합니다.

Python 
t = (10, 20, 30)
t[0]          # 10  (인덱싱 가능)
# t[0] = 99    ← TypeError: 수정 불가!

# 언패킹 (시험 자주 출제)
a, b, c = (1, 2, 3)   # a=1, b=2, c=3
x, *rest = (1,2,3,4)  # x=1, rest=[2,3,4]

# 함수 다중 반환값은 tuple
def minmax(lst):
    return min(lst), max(lst)  # tuple 반환
lo, hi = minmax([3,1,4,1,5])   # lo=1, hi=5
6시험 핵심 요약
  • ⭐list — 순서O, 중복O, 수정O. 음수 인덱스 [-1] = 마지막 원소
  • ⭐dict — 키 중복 불가, 같은 키 재할당 시 덮어쓰기. .get()은 없어도 에러 없음
  • ⭐set — 순서X, 중복X. add()는 이미 있으면 무시. remove()는 없으면 에러, discard()는 안전
  • ⭐tuple — 불변! 수정 시 TypeError. 언패킹으로 여러 변수에 한번에 할당 가능
  • ⭐{} = 빈 dict / 빈 set = set() — 혼동 주의!
