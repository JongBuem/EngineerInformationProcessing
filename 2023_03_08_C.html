Algorithm Study Note

C 팩토리얼(Factorial) 재귀 함수 추적

C · Recursion · Factorial · n! · Base Case · Tracing
Q문제 코드
C 
#include <stdio.h>

int f(int n) {
    if (n < 2) return 1;        // 기저 조건: n=0 또는 n=1 이면 1 반환
    else return n * f(n - 1);    // 재귀: n × f(n-1)
}

int main() {
    printf("%d", f(7));
}

위 코드의 출력값은 무엇인가?

printf() 출력 결과
5040
f(7) = 7! = 7×6×5×4×3×2×1 = 5040
① 문제 유형

팩토리얼(n!)을 재귀로 구현한 함수의 반환값을 추적하는 문제입니다.


  • 🔴기저 조건: n < 2 이면 1 반환 (n=0, n=1 모두 처리)
  • 🔁재귀식: f(n) = n × f(n-1) → 수학적 팩토리얼 정의와 동일
  • 🎯f(7) = 7! = 5040
