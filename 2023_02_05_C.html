Algorithm Study Note

C switch 폴스루(Fall-through) + 정수 나눗셈 추적

C · switch · fall-through · break 없음 · 정수 나눗셈 · Tracing
Q문제 코드
C 
int n[3] = {73, 95, 82};
int sum = 0;

for(i = 0; i < 3; i++){
    sum += n[i];
}

switch(sum / 30){
    case 10:                     // 8≠10 → 스킵
    case  9: printf("A");      // 8≠9  → 스킵
    case  8: printf("B");      // ★ 진입! B 출력
    case  7:                     // fall-through (문장 없음)
    case  6: printf("C");      // fall-through C 출력
    default: printf("D");     // fall-through D 출력
}

위 코드의 출력값은 무엇인가?

출력 결과
BCD
sum=250 → 250/30=8 → case 8 진입
break 없음 → case 7 → case 6(C) → default(D) 연속 실행
① 문제 유형

for 루프 합산 → 정수 나눗셈 → break 없는 switch fall-through를 모두 추적하는 복합 문제입니다.

sum = 73 + 95 + 82 = 250
sum / 30 = 250 / 30 = 8 (정수 나눗셈 — 소수점 버림)
